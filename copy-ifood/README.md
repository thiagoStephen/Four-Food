## Descrição

Aplicativo de delivery baseado no layout do Ifood, em modelo de aprensentação para mobile.

### Dependências

nodejs
npm

### Instalação

```git clone https://github.com/future4code/mello-four-foodB.git ```

```cd mello-four-foodB```

```npm i```

```npm start```
